The Sports Arena Booking System is a full-stack web application designed to simplify sports court reservations and facility management. It enables players to seamlessly search, book, and pay for sports courts, while providing arena owners and administrators with powerful management and analytics tools.

Features
For Players
Browse and search sports courts with location, sport type, and schedule details.
View court availability and book preferred time slots.
Secure online payments via Stripe with automated PDF invoice generation.
Manage upcoming and past bookings through a personalized dashboard.

For Arena Owners
Add, edit, and manage arenas and courts.
Track revenue and booking performance with dynamic charts and reports.
View, approve, or cancel bookings and respond to reviews.
Manage personal profiles and update account details securely.

For Administrators
View and manage user accounts, bookings, and reviews.
Analyze earnings and booking trends using profit analysis tools.
Manage sports categories and pricing rules.

Key Implementations
Role-Based Access Control to manage players, owners, and admin.
Authentication & Security: JWT-based authentication, secure password reset via Nodemailer.
Payments: Stripe payment gateway with automated EJS-based PDF invoices.
Data Visualization: Dynamic reports and charts for revenue tracking and performance analysis.
Responsive Design: Fully responsive UI using React, React-Bootstrap, and EJS templates.

Tech Stack
Frontend: React, React-Bootstrap
Backend: Node.js, Express.js
Database: MySQL
Authentication: JWT
Payments: Stripe API
Email Service: Nodemailer
Templating: EJS
Styling: Bootstrap
